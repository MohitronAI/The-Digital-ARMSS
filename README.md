# The Digital ARMSS

Digital ARMSS is a modern digital marketing and lead management platform built to help businesses manage customer interactions, capture leads, and streamline communication workflows.

## Features

- Contact Form Lead Capture
- Quote Request Management
- Admin Dashboard
- Team Management System
- Role Based Access Control (RBAC)
- Authentication and Authorization
- Lead Tracking and Analytics
- Email Notifications

## Tech Stack

### Frontend
- Next.js 14
- React
- TypeScript
- Tailwind CSS

### Backend
- Next.js API Routes
- Prisma ORM
- PostgreSQL
- NextAuth

### Services
- Resend Email API
- Neon Database
